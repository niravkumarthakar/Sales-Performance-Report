# 📊 Sales Performance Report - Power BI

## 🔹 Project Overview

This project presents an interactive Sales Performance Dashboard built using Microsoft Power BI. It provides a detailed analysis of sales data across regions, product categories, and time periods to help understand business performance and trends.

---

## 🔹 Description

The dashboard transforms raw data into meaningful insights using visual elements such as charts, graphs, and KPI indicators. It helps in identifying high-performing regions, top customers, and profitable product categories. Interactive filters allow users to explore the data dynamically and gain deeper insights.

---

## 🔹 Key Features

* 📌 Total Sales, Profit, and Orders Overview
* 📊 Region-wise and Category-wise Analysis
* 📈 Monthly and Yearly Sales Trends
* 👥 Top Customers Identification
* 🎯 Interactive Filters and Slicers

---
